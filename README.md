# 西华大学学生评价半自动选择评教
该脚本主要在保障教学评价效果的同时提高评价的效率。通过此脚本，用户可以一键选择评价选项、自动填写反馈内容，并快速提交表单。本脚本仅供学习和个人效率提升使用，请勿将其用于任何违反任何规范的用途。

在点击自动评教后悔脚本自动选择第一个问题"比较同意"，其他问题"非常同意"，仅做自动填写（不提交）
